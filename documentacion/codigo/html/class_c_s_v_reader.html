CSVReader Class

Id
csvreader.php 54 2009-10-21 21:01:52Z Pierre-Jean

Allows to retrieve a CSV file content as a two dimensional array. Optionally, the first text line may contains the column names to be used to retrieve fields values (default).

Let's consider the following CSV formatted data: 

   "col1";"col2";"col3"
    "11";"12";"13"
    "21;"22;"2;3"

It's returned as follow by the parsing operation with first line used to name fields: 

    Array(
        [0] => Array(
                [col1] => 11,
                [col2] => 12,
                [col3] => 13
        )
        [1] => Array(
                [col1] => 21,
                [col2] => 22,
                [col3] => 2;3
        )
   )
